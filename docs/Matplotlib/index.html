Python科学计算：Matplotlib¶

本文介绍 Python 的 Matplotlib 绘图库的一些使用方法。全文注重以例说明用法。

读者需要具有：

  • Python 基础

  • 一定程度的 Numpy 基础

  • （可选） Jupyter Notebook 使用经验。

目录:

  • 初识绘图
    • 基础的 plt.plot()
    • 子图绘制 plt.subplots()
    • 保存绘图 f.savefig()
  • 图像控制
    • 默认参数：mpl.rcParams
    • 通用的绘图流程：plt.subplots
    • 图像标题：ax.set_title
    • 坐标轴标题：ax.set_xlabel
    • 图例：ax.legend / f.legend
    • 坐标轴区间：ax.axis / set_xlim
    • 坐标轴刻度
    • 坐标轴刻度网格：ax.grid
    • 坐标轴的高级控制
    • 添加文字：ax.text / annotate
    • 其他坐标系
  • 统计绘图函数
    • 水平或竖直的线/矩形：ax.axhline / axhspan
    • 散点图：ax.scatter
    • 条形图：ax.bar / barh
    • 直方图：ax.hist
    • 核密度曲线
    • 饼图：ax.pie
    • 箱型图：ax.boxplot
